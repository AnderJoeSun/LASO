# IIS 控制器 #

## IIS 硬件连接

### IIS 基本概念
	
	音频总线
	
	1，帧时钟： LRCK (Left/Right)
	
	代表1秒钟时间内有多少个帧数据
	
	典型：wav文件中22Khz，44Khz
	
	22000个音频帧数据，播放1个帧，需要 1/20000 秒 = 50 微秒
	
	
	
	2，位(串行)时钟： SCLK (serial)
	
	代表在1个帧里面，数据所需要的位数
	
	典型：左右声道各8bit，16bit
	
	
	
	结论：SCLK/LRCK = 16 or 32
	
	
	
	3, SD (serial data) 数据线
	
	串行数据
