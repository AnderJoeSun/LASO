# AC97 控制器 #

## AC97 硬件连接
	AC97
	
	
	
	AC97 Controller
	
	------------------
	
	AC-Link
	
	------------------
	
	AC97 CODEC
	
	
	
	
	
	                AC-Link
	
	Controller  ---------------->
	
	               PCM data
	
	
	
	BMP = BMP Header + DIB
	
	WAV = WAV Header + PCM data
	
	
	
	PCM -- DIB 
	
	(Pulse-Code Modulation)
	
	(device independant bitmap)
	
	
	
	
	
	AC97 Signals
	
	------------
	
	1. X97SYNC 帧同步信号
	
	fixed at 48Khz
	
	
	
	2. X97BITCLK 位时钟
	
	fixed at 15.288Mhz
	
	
	
	3. X97SDO/X97SDI 数据输出/输入
	
	
	
	4. Reset
