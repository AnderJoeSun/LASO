### AC97.c 参考代码实现			
	
	AC97 Coding
	
	-------------
	
	1. AC97 Controller Init -> State 3 Ready
	
	2. AC97 CODEC INIT -> Enable, Volume, 44Khz
	
	3. AC97 PCM OUT -> WAV PCM Data -> FIFO Reg
	
	4. AC97 DMA PCM OUT -> DMA
