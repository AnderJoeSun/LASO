## IIS 管脚功能复用
	I2SSDO	:	Serial Data Output 数据输出
	I2SSDI	:	Serial Data Input 数据输入
	I2SSCLK	:	Serial Bit Clock 位时钟
	I2SLRCK	:	Left/Right Channel Clock 帧时钟（左右声道合在一起）
	CDCLK	:	CODEC Clock (System Clock) 编解码器的工作时钟 （最快）fs
