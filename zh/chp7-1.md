# NandFlash 控制器 
## K9F2G08 芯片 
	48-pin TSOP1 封装
	NC - No Connect 不连接，留待扩展 (25pin NC)
	
	控制信号：
	CLE：Command Latch Enable 命令锁存使能
	ALE: Address Latch Enable 地址锁存使能
	nCE: Chip Enable 芯片使能（片选）
	nRE: Read Enable 读使能
	nWE: Write Enable 写使能
	nWP: Write Protect 写保护 
	R/nB: Ready/not Busy 闲/不忙
